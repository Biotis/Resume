## 📂 주요 프로젝트

### 🔹 [Biznum ETL DART Pipeline](https://github.com/sijung-dev/biznum-etl-dart)  
한국 기업 마스터 데이터를 수집하고 전처리 및 검증하는 End-to-End 파이프라인

- **주요 기술**: Python, FastAPI, MongoDB, OpenDART API, NTS API
- **역할**: 전체 설계 및 구현, 데이터 스키마 정의, API 인증 처리
- **특징**:
  - OpenDART와 국세청(NTS) API를 통해 기업 데이터 통합
  - 사업자번호 유효성 검사 포함
  - `.csv`, `.json` 형식으로 export 가능
