### 🔹 [Public-Data-ETL-korea](https://github.com/Biotis/public-data-etl-korea_Douzone_Internship)

한국 기업 데이터 정보를 수집 -> 전처리 -> 사업자등록번호 검증 -> 데이터 형식 변환 프로세스를 자동화한 파이프라인 구축

- **주요 기술**: Python
- **역할**: 전체 설계 및 구현, 데이터 스키마 정의, API 인증 처리
- **특징**:
  - OpenDART와 국세청(NTS) API를 통해 기업 데이터 ETL Process
  - 사업자번호 유효성 검사 포함
  - `.csv`, `.xlsx` 형식으로 export 가능
